# Kafka Addressing VPC Peering Simulation

This example simulates a Kafka broker and multiple clients using Python's standard socket library. It demonstrates how clients attempt to connect to different IP addresses, illustrating the concept of Kafka's internal addressing and how VPC Peering enables secure, direct communication between clients in different virtual networks and a Kafka broker using its internal IP address. It highlights the reachability of internal IPs with and without peering.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
